# Utils 包

## 功能说明

Utils 包提供通用工具函数，包括：

- 文件操作工具
- 路径处理工具
- 字符串处理工具
- 时间处理工具
- 日志工具

## 工具函数示例

### 文件操作
- `CopyFile`: 复制文件
- `MoveFile`: 移动文件
- `DeleteFile`: 删除文件
- `EnsureDir`: 确保目录存在

### 路径处理
- `NormalizePath`: 标准化路径
- `JoinPath`: 连接路径
- `GetFileName`: 获取文件名
- `GetExtension`: 获取文件扩展名
