# LiveBytes

## 项目简介

LiveBytes 是一个基于文件系统交互接口的 AI Agent 框架。

## 项目目标

创建一个智能化的 AI Agent 系统，能够通过统一的工作空间接口并行执行多任务流。

## 架构设计

### 核心理念

LiveBytes 的架构灵感来自于一个简洁而强大的设计思想：**仅使用文件系统目录作为与用户的交互接口**。

### 主要特点

- **文件系统为中心**：所有的用户交互都通过文件系统进行，无需复杂的 API 或 UI 界面
- **统一工作空间**：提供一个统一的工作空间来管理所有任务和数据
- **并行多任务执行**：支持同时执行多个任务流，提高处理效率
- **简单直观**：用户只需要操作文件和目录，就能与 AI Agent 进行交互

### 工作原理

1. **输入**：用户通过在指定目录中创建或修改文件来提交任务
2. **处理**：AI Agent 监控文件系统变化，自动处理新的任务
3. **输出**：处理结果通过文件系统返回给用户
4. **并行执行**：多个任务可以在不同的目录中同时进行
